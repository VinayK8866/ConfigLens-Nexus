# Contributing to ConfigLens

Thank you for your interest in improving ConfigLens! This plugin thrives on community feedback and active collaboration. 

## How You Can Help

1. **Found a bug?** 
   Please **open an issue** detailing the bug, including clear steps to reproduce it, your IDE version, the OS configuration, and stack traces when available.

2. **Have a feature idea?** 
   Open an issue tagged as an `enhancement` and describe what your dream feature looks like. Use cases help us prioritize how to implement it!

3. **Want to contribute code?** 
   - Fork this repository.
   - Create a feature branch (`git checkout -b feature/your-awesome-feature`).
   - Commit your changes (`git commit -m 'Add awesome feature'`). 
     - **Important:** Make sure any new Java files start with the standard [Eclipse Public License v2.0 (EPL)](http://www.eclipse.org/legal/epl-v20.html) legal header block.
   - Push to your branch and open a Pull Request against the `main` branch.

4. **Help with documentation:** 
   Feel free to add tutorials, fix typos, or expand the wiki to make it easier for new users.

5. **Discussions:** 
   Jump into issue threads, help other users, and share your experiences using ConfigLens!
