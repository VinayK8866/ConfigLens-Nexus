🚀 ConfigLens is Ready!

Thank you for installing ConfigLens. Your Eclipse IDE now has high-performance intelligence for YAML, JSON, and .env files.

🛡️ Real-time Secret Detection ConfigLens scans your files for 20+ patterns (AWS keys, tokens, etc.) and warns you before you commit them.
👻 Ghost Text Previews See the actual values of your ${ENV_VARS} right inside the editor as non-intrusive gray text.
🧭 Interactive Breadcrumbs Never lose your place. Top-level sticky navigation helps you drill into deep configuration trees.
⚡ Flat View Open Window > Show View > Flat View to see your deep config files represented as a searchable key-value table.

💡 Pro Tips

To ignore a false-positive secret warning, simply add # configlens-ignore to the end of the line.

Right-Click Actions: You can now right-click any property name to copy its Dot-Notation Path (e.g., database.url) or JSONPath.

Safe for Big Files: ConfigLens uses a streaming, interruptible parser. It can handle 100MB+ files without slowing down your computer.
